## 🚀 启动方式

### 快速启动 (Windows)
```bash

```

### 手动启动
```bash
# 1. 安装依赖
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt

# 2. 配置环境变量
copy .env.example .env
# 编辑.env填入API密钥

# 3. 启动Web UI
python stock_analysis_assistant.py

# 4. (可选)启动定时任务
python scheduler.py
```
